Command Line Usage

The command line executable version has the following options:

ppxml [options] [input] [output]
-i:num Specifies indent level (default 4).
-c:numSpecifies space char (32= space,9=tab), (default=32).
-e:encodingSpecifies the encoding to use for output (default UTF8).
-a Causes attributes to be formatted each on their own line. This makes attribute heavy XML more readable.
input Input file or URL, default is stdin.
output

Output file (cannot be same as input), default is stdout
