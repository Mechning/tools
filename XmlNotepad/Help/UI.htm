User Interface

The interface consists of the following components:

Menu bar
See Menus for description of each menu.
Tool Bar
The toolbar provides some shortcuts for commonly used menu commands, hover over the toolbar buttons to see a popup showing the menu command they invoke.
Tree View
This view shows the XML document hierarchy in a tree showing element, attribute and processing instructions names as well as standard names for comment, text, CDATA, whitespace, and significant-whitespace nodes.
Node Value View
This view shows the text value of the adjacent node in the tree view. For element nodes it normally shows a gray band unless the element contains only one text node child in which case it displays that text value.
Error List
The error list shows validation errors and warnings.
Status Bar
The status bar shows various messages indicating the state of the program.
Resizers
You can move the resizers to change the ratio of space allocated to each of the main three panels.

See also Keyboard Interface.
